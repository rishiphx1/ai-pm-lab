# AI PM Lab Workspace

This workspace is a structured home for product thinking, experimentation, and delivery planning.

## Folder guide

- `docs/` - Core strategy and product documents.
- `templates/` - Reusable templates for PM workflows.
- `notes/` - Working notes from calls, interviews, and ideas.
- `research/` - Research summaries, market scans, and analysis.
- `sprints/` - Sprint plans, retros, and execution tracking.

## Quick start

1. Fill `docs/product-vision.md` with the problem and target user.
2. Write your current quarter goals in `docs/roadmap.md`.
3. Use templates in `templates/` to standardize planning.
4. Create weekly execution notes in `sprints/`.
